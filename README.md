# Harmonius

Harmonius is a modern cross-platform graphics framework for building real-time
2D/3D rendering and data-parallel computing applications.

Harmonius supports desktop applications on MacOS, Linux, and Windows. Mobile
applications on iOS and Android are also supported. Harmonius's backend can use
Metal 4, Direct3D 12, or Vulkan 1.4 (with extensions).

Harmonius does not support the traditional rendering pipeline or any legacy API
backends. Harmonius is designed for GPU-driven renderer architectures that use
indirect rendering. This reduces the reach of the project, but it reduces scope
more.
