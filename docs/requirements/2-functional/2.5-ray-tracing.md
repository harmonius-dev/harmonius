# 2.5 Ray Tracing

## R-2.5.1 Query RT Hardware Availability

The system SHALL expose a capability descriptor indicating which ray tracing features are supported by the current hardware. Applications SHALL query this before enabling RT effects.

## R-2.5.2 Enable RT Acceleration Structures

The system SHALL automatically build and compact bottom-level acceleration structures (BLAS) from mesh geometry and refit the top-level acceleration structure (TLAS) each frame for dynamic scenes.

- **Features:** [F-2.1.1](../../features/2-advanced-rendering/2.1-ray-tracing.md) acceleration structures

## R-2.5.3 Configure RT Reflections

The system SHALL support hybrid SSR+RT reflections with a configurable roughness threshold determining when RT rays supplement screen-space ray marching.

- **Features:** [F-2.1.2](../../features/2-advanced-rendering/2.1-ray-tracing.md) RT reflections

## R-2.5.4 Select Reflection Fallback

The system SHALL support selecting a reflection fallback mode: SSR-only, cubemap probes, or disabled when RT hardware is unavailable.

## R-2.5.5 Configure RT Sample Counts

The system SHALL allow configuring rays-per-pixel independently for each RT effect (reflections, shadows, GI, AO).

## R-2.5.6 Configure Denoiser Strength

The system SHALL expose per-effect spatial-temporal denoiser aggressiveness controls. Higher aggressiveness SHALL reduce noise at the cost of temporal lag and detail softening.

## R-2.5.7 Mark Entities for RT Participation

The system SHALL support per-entity flags controlling inclusion or exclusion from the RT acceleration structure.

## R-2.5.8 Enable RT Global Illumination

The system SHALL support DDGI-based real-time global illumination with configurable probe update rate and hysteresis for temporal stability.

- **Features:** [F-2.1.4](../../features/2-advanced-rendering/2.1-ray-tracing.md) DDGI

## R-2.5.9 Enable Path Tracing

The system SHALL support a progressive unbiased path tracer with configurable samples per pixel and maximum bounce count. The path tracer SHALL support all material types, volumetrics, and sky atmosphere for ground-truth reference output.

- **Features:** [F-2.1.5](../../features/2-advanced-rendering/2.1-ray-tracing.md) path tracing

## R-2.5.10 Enable RT Subsurface Transmission

The system SHALL support volumetric scattering through translucent media during shadow ray evaluation. When a shadow ray hits a subsurface material, light transport through the material's thickness SHALL be evaluated using the subsurface profile.

- **Features:** [F-2.1.6](../../features/2-advanced-rendering/2.1-ray-tracing.md) RT subsurface transmission
