# 2.17 VFX and Particles

## R-2.17.1 Create GPU Particle Emitters

The system SHALL support creating compute-shader particle emitters with configurable spawn rate, lifetime, and a chain of simulation modules (forces, noise, collision, drag) evaluated per-particle per-frame.

- **Features:** [F-7.1.1](../../features/7-vfx/7.1-particle-rendering.md) GPU particle simulation

## R-2.17.2 Configure Sprite Particles

The system SHALL support camera-facing billboard particle rendering with atlas-based flipbook animation, configurable blend modes (additive, alpha, premultiplied), and per-particle color, size, and rotation curves.

- **Features:** [F-7.1.2](../../features/7-vfx/7.1-particle-rendering.md) sprite particle rendering

## R-2.17.3 Configure Mesh Particles

The system SHALL support instanced 3D mesh rendering driven by particle state with full material support, per-particle transform, scale, and material parameter overrides.

- **Features:** [F-7.1.3](../../features/7-vfx/7.1-particle-rendering.md) mesh particle rendering

## R-2.17.4 Configure Ribbon/Trail Particles

The system SHALL support spline-based ribbon geometry connecting sequential particles with configurable width, twist, UV tiling, and tessellation.

- **Features:** [F-7.1.4](../../features/7-vfx/7.1-particle-rendering.md) ribbon and trail rendering

## R-2.17.5 Enable Particle Light Emission

The system SHALL support dynamic point lights spawned from particle positions with configurable color, intensity, and attenuation radius contributing to the scene's light buffer.

- **Features:** [F-7.1.5](../../features/7-vfx/7.1-particle-rendering.md) particle light emission

## R-2.17.6 Configure GPU Fluid Simulation

The system SHALL support grid-based Eulerian fluid simulation on the GPU for fire, smoke, and liquid effects. The density field SHALL be ray-marched with emission, absorption, and scattering.

- **Features:** [F-7.1.6](../../features/7-vfx/7.1-particle-rendering.md) GPU fluid simulation

## R-2.17.7 Configure Particle Collision

The system SHALL support GPU-driven particle collision against the scene depth buffer (screen-space) and signed distance fields (world-space) with configurable restitution and friction parameters.

- **Features:** [F-7.1.7](../../features/7-vfx/7.1-particle-rendering.md) particle collision

## R-2.17.8 Configure Particle Data Interfaces

The system SHALL support typed data channels allowing particles to read from and write to external data sources: skeletal mesh surfaces, textures, curves, and other emitters.

- **Features:** [F-7.1.8](../../features/7-vfx/7.1-particle-rendering.md) particle data interfaces

## R-2.17.9 Configure Particle Warm-Up

The system SHALL support pre-simulating emitters forward to a target time before the first visible frame, ensuring particles appear fully populated.

## R-2.17.10 Configure Particle LOD

The system SHALL support distance-based emitter scaling that reduces spawn rate, simplifies simulation modules, and culls emitters at configurable distance thresholds.

## R-2.17.11 Enable Destruction System

The system SHALL support real-time destruction using volume distance fields for destruction masking and fracture pattern generation. Pre-fractured mesh chunks SHALL activate on damage events with physics simulation, cascading structural failure, and debris particle spawning.

- **Features:** [F-7.1.9](../../features/7-vfx/7.1-particle-rendering.md) destruction system
