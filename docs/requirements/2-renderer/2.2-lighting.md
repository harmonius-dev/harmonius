# 2.2 Lighting

## R-2.2.1 Create Point Lights

The system SHALL support creating point lights with configurable position, color, intensity, and attenuation radius. Point lights SHALL contribute to both forward+ and deferred lighting paths.

- **Features:** [F-1.1.1](../../features/1-rendering/1.1-core-rendering.md) direct lighting

## R-2.2.2 Create Spot Lights

The system SHALL support creating spot lights with configurable position, direction, color, intensity, inner and outer cone angles, and attenuation radius.

- **Features:** [F-1.1.1](../../features/1-rendering/1.1-core-rendering.md) direct lighting

## R-2.2.3 Create Directional Lights

The system SHALL support creating directional lights with configurable direction, color, and intensity for sun and moon illumination.

- **Features:** [F-1.1.1](../../features/1-rendering/1.1-core-rendering.md) direct lighting

## R-2.2.4 Create Area Lights

The system SHALL support rectangular and spherical area lights with LTC (linearly-transformed cosine) integration. Source size SHALL be configurable, producing physically correct soft reflections and falloff.

- **Features:** [F-1.1.10](../../features/1-rendering/1.1-core-rendering.md) area lights

## R-2.2.5 Configure Sky Light (IBL)

The system SHALL capture the sky atmosphere or a provided cubemap and pre-filter it into diffuse irradiance and split-sum specular lookup tables for image-based lighting.

- **Features:** [F-1.1.11](../../features/1-rendering/1.1-core-rendering.md) sky light

## R-2.2.6 Assign IES Light Profiles

The system SHALL support loading photometric data from IES files and assigning profiles to point and spot lights. The intensity distribution SHALL modulate the light's contribution as a function of angle.

- **Features:** [F-1.1.12](../../features/1-rendering/1.1-core-rendering.md) IES light profiles

## R-2.2.7 Configure Light Functions

The system SHALL support assigning material-driven intensity and color modulation masks to any light. A lightweight material graph SHALL produce a scalar or color mask evaluated per-pixel within the light's influence volume.

- **Features:** [F-1.1.13](../../features/1-rendering/1.1-core-rendering.md) light functions

## R-2.2.8 Configure Forward+ Tile Size

The system SHALL allow configuring the screen-space tile size for the tiled/clustered light assignment grid. Default tile size SHALL be 16x16 pixels.

- **Features:** [F-1.2.1](../../features/1-rendering/1.2-lighting-and-materials.md) forward+ lighting

## R-2.2.9 Observe Active Light Count

The system SHALL expose the current active light count as a queryable diagnostic. A warning SHALL be emitted when the count approaches the configured light budget.

## R-2.2.10 Configure DDGI Probe Grid

The system SHALL allow configuring DDGI probe grid spacing, dimensions, and per-probe texel resolution. The grid SHALL support up to 64x32x64 probes.

- **Features:** [F-2.1.4](../../features/2-advanced-rendering/2.1-ray-tracing.md) DDGI

## R-2.2.11 Toggle DDGI at Runtime

The system SHALL allow enabling and disabling DDGI at runtime. When disabled, an ambient fallback term SHALL provide baseline indirect lighting.

- **Features:** [F-2.1.4](../../features/2-advanced-rendering/2.1-ray-tracing.md) DDGI

## R-2.2.12 Configure RT Indirect Lighting

The system SHALL allow configuring the sample count and bounce limit for ray-traced indirect lighting independently of other RT effects.

- **Features:** [F-2.1.3](../../features/2-advanced-rendering/2.1-ray-tracing.md) RT indirect lighting

## R-2.2.13 Configure God Rays

The system SHALL allow designating light sources as god ray emitters and selecting between screen-space radial blur and volumetric integration modes.

- **Features:** [F-2.2.4](../../features/2-advanced-rendering/2.2-environment.md) god rays

## R-2.2.14 Adjust Light Priority

The system SHALL support per-light priority weights that influence shadow atlas tile allocation. Higher-priority lights SHALL receive larger shadow map tiles.

## R-2.2.15 Enable Stochastic Many-Light Sampling

The system SHALL support importance-sampled direct lighting for scenes with thousands of shadowed area lights. A fixed ray budget per pixel SHALL be distributed across light sources proportional to estimated contribution, with temporal accumulation denoising.

- **Features:** [F-1.2.9](../../features/1-rendering/1.2-lighting-and-materials.md) stochastic many-light sampling
