# Rendering Requirements

Requirements for the rendering subsystem including the renderer, render graph library, and GPU
runtime layer.

## Files

| File | Scope |
|------|-------|
| [renderer.md](renderer.md) | High-level renderer requirements (R-2.x.y) -- index of all renderer functional requirements |
| [render-graph.md](render-graph.md) | Render graph library requirements (RG-x.y) -- 119 requirements across 14 categories |
| [gpu-abstraction.md](gpu-abstraction.md) | GPU runtime requirements (GR-x.y) -- 36 requirements across 4 categories |

## Traceability

The render graph requirements were derived from the high-level renderer requirements via systematic
translation and deduplication. See
[traceability-matrix.md](../6-render-graph/traceability-matrix.md) for the complete bidirectional
mapping.
