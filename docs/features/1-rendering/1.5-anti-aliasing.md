# 1.5 Anti-Aliasing and Upscaling

## F-1.5.1 Temporal Anti-Aliasing (TAA)

Multi-frame jittered sample accumulation that blends the current frame with reprojected history using motion vectors. Reduces geometric aliasing and specular shimmer. Temporal weighting clamps reject disoccluded pixels to reduce ghosting.

## F-1.5.2 Temporal Super Resolution (TSR)

Temporal upscaler rendering high-resolution output from a lower internal resolution (e.g., 1080p to 4K). Combines sub-pixel jitter accumulation with a learned or heuristic detail reconstruction filter. A platform-agnostic compute shader implementation with quality/performance scalability controls.

## F-1.5.3 FXAA (Fast Approximate Anti-Aliasing)

Single-pass spatial anti-aliasing using high-contrast edge detection and directional pixel blending. Lightweight post-process with minimal performance cost and no temporal history dependency.

## F-1.5.4 MSAA (Multi-Sample Anti-Aliasing)

Hardware multi-sample anti-aliasing sampling multiple sub-pixel locations per fragment during rasterization. Available only in the forward rendering path since deferred G-buffer layout is incompatible with per-sample storage. Supports 2x and 4x sample counts.
