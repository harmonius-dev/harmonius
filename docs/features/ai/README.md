# AI Features

AI systems covering navigation, steering, decision-making, perception, and crowd simulation for
MMO-scale open worlds with thousands of server-side NPCs.

## Files

| File | Scope |
|------|-------|
| [navigation.md](navigation.md) | NavMesh generation, tiling/streaming, A* pathfinding, string pulling, path smoothing, dynamic obstacles, off-mesh links, runtime tile rebuild, background generation, destruction integration, player structures, multi-agent meshes, dynamic area costs, HPA*, debug visualization |
| [steering-avoidance.md](steering-avoidance.md) | RVO/ORCA local avoidance, steering behaviors, formation movement, group steering |
| [behavior-trees.md](behavior-trees.md) | Composite/decorator/leaf nodes, blackboard, tree assets, conditional aborts, debugging |
| [utility-ai.md](utility-ai.md) | Scoring functions, response curves, action selection, dual utility, context-based reasoning |
| [goap.md](goap.md) | GOAP planner, world state, action preconditions & effects, plan caching, replanning |
| [perception.md](perception.md) | Sight, hearing, damage sense, faction awareness, stimuli management, sense aging |
| [crowd-simulation.md](crowd-simulation.md) | Flocking, flow fields, mass entity simulation, AI LOD, density management |
