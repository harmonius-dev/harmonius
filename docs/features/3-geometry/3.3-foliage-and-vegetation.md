# 3.3 Foliage and Vegetation

## F-3.3.1 Hierarchical Instanced Rendering

Hardware instancing with a hierarchical spatial tree for dense vegetation. Instance data (transform, per-instance random seed, LOD) is stored in a GPU buffer. The spatial tree enables efficient frustum and distance culling of large instance populations (millions of plants) with minimal CPU overhead.

- **Requirements:** [R-1.1.2](../../requirements/1-architecture/1.1-core-constraints.md) GPU-driven

## F-3.3.2 Foliage Wind Animation

GPU-driven per-instance wind deformation applied in the vertex/mesh shader. A global wind field combined with per-instance phase offsets produces natural swaying. Supports hierarchical wind layers: trunk sway, branch oscillation, and leaf flutter, each with independent frequency and amplitude.

## F-3.3.3 Procedural Foliage Distribution

Rule-based ecosystem placement generating foliage instance positions from density maps, biome rules, and species interaction constraints. Species compete for space based on configurable shade tolerance, moisture requirements, and growth patterns to produce naturalistic distributions.

## F-3.3.4 Per-Instance Fade

Vertex-shader-computed distance fade that gradually reduces instance opacity before the cull distance threshold. Each instance interpolates between fully opaque and fully transparent over a configurable fade range, eliminating hard pop-in at LOD and cull boundaries.

## F-3.3.5 Landscape Grass System

Procedural grass generation tied to terrain material layers. Grass mesh instances are automatically spawned based on landscape layer weight maps, with configurable density, mesh variety, and placement randomization. Instances are generated on demand within the camera's view distance.

## F-3.3.6 Two-Sided Foliage Transmission

Dedicated foliage shading supporting light transmission through thin organic surfaces. Back-facing fragments receive transmitted light colored by a subsurface color parameter, enabling natural backlit leaf and petal rendering without the cost of full subsurface scattering.
