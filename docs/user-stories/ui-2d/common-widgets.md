1. **US-10.2.1** — **As a** player (P-23), **I want** rich text with inline formatting, icons, and
   hyperlinks in chat and tooltips, **so that** I can read styled content in any language.
2. **US-10.2.2** — **As a** player (P-23), **I want** text input with IME support for CJK languages,
   clipboard, and undo/redo, **so that** I can compose chat in any language.
3. **US-10.2.3** — **As a** player (P-23), **I want** responsive buttons, sliders, checkboxes, and
   toggles with animated state transitions, **so that** settings interaction is smooth.
4. **US-10.2.4** — **As a** player (P-23), **I want** searchable dropdown menus that filter as I
   type, **so that** I find options quickly in long lists.
5. **US-10.2.5** — **As a** player (P-23), **I want** virtualized list views with inertial scrolling
   that handle thousands of entries, **so that** browsing large lists feels responsive.
6. **US-10.2.6** — **As a** player (P-23), **I want** hover-triggered tooltips, right-click context
   menus, and modal confirmation dialogs, **so that** I can inspect and manage items.
7. **US-10.2.7** — **As a** player (P-23), **I want** to drag and drop items between inventory,
   equipment, action bars, and trade windows, **so that** I manage items intuitively.
8. **US-10.2.8** — **As a** game designer (P-5), **I want** tooltip layouts with stat comparisons
   and set bonuses, **so that** players make informed decisions.
9. **US-10.2.9** — **As a** player (P-23), **I want** determinate progress bars and loading spinners
   showing accurate progress, **so that** I know how long operations take.
10. **US-10.2.10** — **As a** technical artist (P-13), **I want** to customize progress bar fills,
    gradients, and label fonts, **so that** bars match the game's visual design.
