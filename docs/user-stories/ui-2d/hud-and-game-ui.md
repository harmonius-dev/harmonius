1. **US-10.3.1** — **As a** player (P-23), **I want** health, mana, and cast bars with animated
   fill, predicted damage overlays, and absorb shields, **so that** I monitor status at a glance in
   combat.
2. **US-10.3.2** — **As a** technical artist (P-13), **I want** to customize bar segmentation, fill
   colors, and border styles per resource type, **so that** bars match the game's visual identity.
3. **US-10.3.3** — **As a** player (P-23), **I want** buff/debuff icons with radial cooldown sweeps,
   countdown text, and tooltip inspection, **so that** I track 30+ effects during raids.
4. **US-10.3.4** — **As a** game designer (P-5), **I want** to configure how buffs are grouped,
   prioritized, and filtered, **so that** the most important effects are visible.
5. **US-10.3.5** — **As a** player (P-23), **I want** action bars with frame-accurate cooldown
   sweeps, charge counts, and keybind labels, **so that** I know exactly when abilities are ready.
6. **US-10.3.6** — **As a** player (P-23), **I want** floating nameplates above characters with
   overlap avoidance and distance culling, **so that** I identify entities in crowded areas.
7. **US-10.3.7** — **As a** game designer (P-5), **I want** configurable damage number fonts, colors
   per damage type, and merge behavior, **so that** combat text fits the game's style.
8. **US-10.3.8** — **As a** player (P-23), **I want** a minimap with real-time icons and a world map
   with zoom, pan, fog-of-war, and waypoints, **so that** I navigate without getting lost.
9. **US-10.3.9** — **As a** player (P-23), **I want** a quest tracker with objectives, waypoint
   markers, and compass indicators, **so that** I always know where to go next.
10. **US-10.3.10** — **As a** game designer (P-5), **I want** map markers defined as data assets
    with icon, category, and visibility rules, **so that** all markers are managed uniformly.
11. **US-10.3.11** — **As a** player (P-23), **I want** a compass strip with tracked markers that
    rotate with my facing, **so that** I maintain directional awareness.
12. **US-10.3.12** — **As a** player (P-23), **I want** off-screen indicators pointing toward
    objectives outside the viewport, **so that** I know which direction to turn.
13. **US-10.3.13** — **As a** player (P-23), **I want** a chat window with multiple channels, tabs,
    clickable links, and emoji, **so that** I communicate effectively in busy areas.
14. **US-10.3.14** — **As a** game designer (P-5), **I want** a single grid widget that adapts to
    bags, bank, vendor, trade, and loot contexts, **so that** all containers share consistent
    behavior.
15. **US-10.3.15** — **As a** player (P-23), **I want** quest markers on minimap, compass, and world
    map simultaneously with consistent icons, **so that** I find objectives from any view.
16. **US-10.3.16** — **As a** game developer (P-15), **I want** procedural minimap generation from
    terrain, biome, and road data without artist intervention, **so that** generated worlds have
    minimaps instantly.
17. **US-10.3.17** — **As a** player (P-23), **I want** the world map to support post-processing
    styles (parchment, painted, tactical), **so that** the map reinforces the game's aesthetic.
18. **US-10.3.18** — **As a** technical artist (P-13), **I want** to layer hand-painted overlays on
    the generated map while preserving dynamic markers, **so that** key zones have maximum artistic
    quality.
