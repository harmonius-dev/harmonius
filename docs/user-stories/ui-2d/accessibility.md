# User Stories — 10.6 Accessibility

## US-10.6.1 Play the Game Regardless of Visual, Auditory, or Motor Ability
**As a** player, **I want** screen reader support, subtitles, colorblind modes, scalable
high-contrast UI, full input remapping, and text-to-speech for chat, **so that** I can
enjoy the game fully regardless of my disability.

| Acceptance criteria | Features | Requirements |
|---|---|---|
| Screen reader announces all widget labels and states | F-10.6.1 | R-10.6.1 |
| Configurable subtitles and closed captions | F-10.6.2 | R-10.6.2 |
| Protanopia, deuteranopia, tritanopia color filters | F-10.6.3 | R-10.6.3 |
| High contrast mode and 80%-250% UI scaling | F-10.6.4 | R-10.6.4 |
| Full input remapping with one-handed and switch input | F-10.6.5 | R-10.6.5 |
| Per-channel text-to-speech and speech-to-text | F-10.6.6 | R-10.6.6 |
| WCAG 2.1 AA for menus, AAA for critical gameplay | F-10.6.7 | R-10.6.7 |
| Cross-domain accessibility hooks (audio, physics, AI) | F-10.6.1 | R-X.9.6 |
