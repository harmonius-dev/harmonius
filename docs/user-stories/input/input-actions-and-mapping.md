1. **US-6.2.1.1** — define boolean actions (pressed/not pressed)
   - **Acceptance:** simple triggers like "Jump" and "Dodge" work
2. **US-6.2.1.2** — define axis 1D actions (scalar float)
   - **Acceptance:** throttle and zoom inputs produce continuous values
3. **US-6.2.1.3** — define axis 2D actions (vector2)
   - **Acceptance:** movement and camera look produce directional input
4. **US-6.2.1.4** — implement strongly typed actions decoupling gameplay from device inputs
   - **Acceptance:** the same action fires from any device
5. **US-6.2.1.5** — the same game actions to work on keyboard, gamepad, and touch
   - **Acceptance:** I can use any device
6. **US-6.2.1.6** — axis 3D actions (vector3)
   - **Acceptance:** VR motion and spatial input produce 3-axis values
7. **US-6.2.1.7** — touch virtual joystick as the default axis2D source on mobile
   - **Acceptance:** movement works on touchscreens

   sources
   - **Acceptance:** typed actions work
