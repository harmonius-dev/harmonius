# User Stories: World Building

## US-15.6.1 Terrain Sculpting at Scale

**As a** designer, **I want** to sculpt massive open-world terrains with raise, smooth,
erode, and flatten brushes that stream to disk incrementally, **so that** I can shape
landscapes of any size without loading the entire heightmap into memory.

## US-15.6.2 Rule-Based Vegetation Placement

**As a** designer, **I want** to define biome rules with per-species slope, altitude, and
clustering constraints and auto-populate large regions, **so that** I can cover square
kilometers with ecologically consistent vegetation and then hand-refine specific areas.
