# User Stories: Build and Deployment

## US-15.14.1 One-Click Platform Packaging

**As a** DevOps engineer, **I want** to package the game into platform-native formats
(macOS .app, Windows .exe, iOS IPA, Android APK, console packages) from the editor UI or
CLI, **so that** builds are reproducible and integrated into CI pipelines.

## US-15.14.2 Deploy-to-Device Iteration

**As a** developer, **I want** one-click deployment to connected development devices with
incremental asset transfer, **so that** I can test changes on target hardware within
seconds instead of performing a full rebuild and manual file copy.
