# User Stories: Engine Launcher

## US-15.15.1 Side-by-Side Engine Versions

**As a** developer, **I want** to install and manage multiple engine versions side by side
with version-pinned projects, **so that** I can maintain legacy projects on older versions
while evaluating new releases without version conflicts.

## US-15.15.2 Guided Project Creation

**As a** designer, **I want** a creation wizard that lets me pick a genre template (RPG,
FPS, RTS, 2D platformer, VR), target platforms, and starter content, **so that** new
projects start with pre-configured gameplay systems and input mappings.
