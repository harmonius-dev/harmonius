# User Stories: Remote Editing and Collaboration

## US-15.12.1 Real-Time Collaborative Editing

**As a** designer, **I want** to edit the same world simultaneously with teammates, seeing
their cursors and edits in real time via CRDT synchronization, **so that** the team can
work on the same level without locking files or merging offline changes.

## US-15.12.2 Remote GPU Session Handoff

**As a** developer, **I want** to suspend a remote editing session at the office and resume
it from home on a different device with all panels, camera positions, and unsaved work
intact, **so that** I can transition between locations without losing context.
