# User Stories: Editor Framework

## US-15.1.1 Custom Workspace Layout

**As a** designer, **I want** to dock, split, tab, and float editor panels into a saved
workspace layout, **so that** I can switch instantly between level design and material
authoring workflows without rearranging panels each time.

## US-15.1.2 Undo Grouped Operations

**As an** artist, **I want** to undo a multi-entity move as a single step, **so that** I
can revert accidental batch transformations without clicking undo once per entity.
