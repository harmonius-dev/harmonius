# User Stories: Version Control and Git Integration

## US-15.10.1 In-Editor Git Workflow

**As a** designer, **I want** to stage, commit, push, branch, and create pull requests
entirely within the editor UI, **so that** I can manage version control without switching
to a command-line terminal or external Git client.

## US-15.10.2 Structural Asset Merge

**As a** developer, **I want** the Git merge driver to perform three-way structural merge
on binary assets like logic graphs and prefabs, **so that** concurrent edits by teammates
are resolved automatically at the node and property level instead of failing as binary
conflicts.
