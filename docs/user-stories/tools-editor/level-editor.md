# User Stories: Level Editor

## US-15.2.1 Surface Snap Placement

**As a** designer, **I want** to drag entities from the asset browser and snap them onto
terrain or mesh surfaces, **so that** I can populate environments quickly without manually
entering Y coordinates for each object.

## US-15.2.2 Nested Prefab Propagation

**As a** designer, **I want** changes to a parent prefab to propagate to all nested
instances across the world unless I have overridden a property, **so that** I can update
hundreds of village layouts by editing a single house prefab.
