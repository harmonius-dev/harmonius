# User Stories -- 15.22 VR Editor Mode

## Stories

| ID           | Persona                 |
|--------------|-------------------------|
| US-15.22.1.1 | level designer (P-6)    |
| US-15.22.1.2 | game designer (P-5)     |
| US-15.22.2.1 | level designer (P-6)    |
| US-15.22.2.2 | game designer (P-5)     |
| US-15.22.3.1 | level designer (P-6)    |
| US-15.22.3.2 | game designer (P-5)     |
| US-15.22.4.1 | game designer (P-5)     |
| US-15.22.4.2 | level designer (P-6)    |
| US-15.22.5.1 | level designer (P-6)    |
| US-15.22.5.2 | game designer (P-5)     |
| US-15.22.6.1 | game designer (P-5)     |
| US-15.22.6.2 | level designer (P-6)    |

1. **US-15.22.1.1** — **As a** level designer (P-6), **I want** the editor rendered in a VR headset
   with stereoscopic rendering and head tracking, **so that** I experience my level at true player
   scale.

2. **US-15.22.1.2** — **As a** game designer (P-5), **I want** VR mode to share world state with the
   desktop editor, **so that** changes are visible on both displays.

3. **US-15.22.2.1** — **As a** level designer (P-6), **I want** to grab, move, rotate, and scale
   objects with motion controllers and haptic feedback, **so that** spatial editing feels natural.

4. **US-15.22.2.2** — **As a** game designer (P-5), **I want** snap-to-grid with trigger-based
   precision placement, **so that** VR-placed objects align to the design grid.

5. **US-15.22.3.1** — **As a** level designer (P-6), **I want** bare-hand editing with pinch, grab,
   and point gestures on headsets with optical hand tracking, **so that** I edit without
   controllers.

6. **US-15.22.3.2** — **As a** game designer (P-5), **I want** seamless switching between hand
   tracking and controllers, **so that** I use whichever input is convenient.

7. **US-15.22.4.1** — **As a** game designer (P-5), **I want** a spatial UI with floating panels,
   radial menus, and gaze-activated widgets, **so that** tools are accessible in VR.

8. **US-15.22.4.2** — **As a** level designer (P-6), **I want** text input via virtual keyboard or
   voice dictation, **so that** I can rename entities without removing my headset.

9. **US-15.22.5.1** — **As a** level designer (P-6), **I want** to see other users' avatars and
   cursor positions in VR during collaboration, **so that** I know where teammates are working.

10. **US-15.22.5.2** — **As a** game designer (P-5), **I want** spatialized voice chat positioned at
    each user's avatar, **so that** communication feels natural in the shared space.

11. **US-15.22.6.1** — **As a** game designer (P-5), **I want** to follow another user's or AI
    agent's viewpoint in VR, **so that** I can observe their work or review their edits.

12. **US-15.22.6.2** — **As a** level designer (P-6), **I want** follow mode to buffer camera motion
    to prevent VR discomfort, **so that** following fast-moving users is safe.
