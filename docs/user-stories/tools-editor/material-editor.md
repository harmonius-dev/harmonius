# User Stories: Material Editor

## US-15.3.1 Visual Shader Authoring

**As an** artist, **I want** to author materials by connecting nodes in a visual graph with
type-safe pins and instant 3D preview, **so that** I can create shaders without writing
code and see the result immediately.

## US-15.3.2 Material Instance Variation

**As an** artist, **I want** to create lightweight material instances that override only
specific parameters of a parent material, **so that** I can produce thousands of color and
weathering variations without duplicating shaders or consuming extra GPU memory.
