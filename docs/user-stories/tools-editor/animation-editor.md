# User Stories: Animation Editor

## US-15.4.1 Keyframe Curve Tuning

**As an** artist, **I want** to edit animation curves with Bezier tangent handles and
preset ease modes, **so that** I can fine-tune interpolation between keyframes until the
motion feels natural.

## US-15.4.2 Cross-Rig Retargeting

**As an** artist, **I want** to map bone pairs between source and target skeletons and
preview retargeted animation in real time, **so that** I can share animation libraries
across humanoid, creature, and mount rigs without re-authoring each clip.
