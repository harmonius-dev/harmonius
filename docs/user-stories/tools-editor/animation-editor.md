# User Stories -- 15.4 Animation Editor

## Stories

| ID          | Persona                 |
|-------------|-------------------------|
| US-15.4.1.1 | game designer (P-5)     |
| US-15.4.1.2 | technical artist (P-13) |
| US-15.4.2.1 | technical artist (P-13) |
| US-15.4.2.2 | game designer (P-5)     |
| US-15.4.3.1 | technical artist (P-13) |
| US-15.4.3.2 | game designer (P-5)     |
| US-15.4.4.1 | game designer (P-5)     |
| US-15.4.4.2 | technical artist (P-13) |
| US-15.4.5.1 | game designer (P-5)     |
| US-15.4.5.2 | technical artist (P-13) |
| US-15.4.6.1 | game designer (P-5)     |
| US-15.4.6.2 | technical artist (P-13) |
| US-15.4.7.1 | technical artist (P-13) |
| US-15.4.7.2 | game designer (P-5)     |

1. **US-15.4.1.1** — **As a** game designer (P-5), **I want** a multi-track animation timeline with
   playback controls and scrubbing, **so that** I can review and edit keyframed animations.

2. **US-15.4.1.2** — **As a** technical artist (P-13), **I want** color-coded tracks for
   translation, rotation, scale, and custom properties, **so that** I can distinguish channels at a
   glance.

3. **US-15.4.2.1** — **As a** technical artist (P-13), **I want** a curve editor with Bezier and
   Hermite tangent modes and multi-curve batch editing, **so that** I can fine-tune easing across
   many bones simultaneously.

4. **US-15.4.2.2** — **As a** game designer (P-5), **I want** curve presets for ease-in, ease-out,
   linear, and stepped, **so that** I can apply standard timing without manual tangent adjustment.

5. **US-15.4.3.1** — **As a** technical artist (P-13), **I want** a skeleton viewer with selectable
   bones showing constraints and IK chains, **so that** I can inspect rig hierarchies.

6. **US-15.4.3.2** — **As a** game designer (P-5), **I want** toggling between bone display modes
   and overlaying multiple skeletons for comparison, **so that** I can verify retargeting setups.

7. **US-15.4.4.1** — **As a** game designer (P-5), **I want** animation playback on a character mesh
   in a preview viewport with orbit camera, **so that** I can evaluate motion quality.

8. **US-15.4.4.2** — **As a** technical artist (P-13), **I want** debug overlays showing velocity
   vectors, contact points, and bone trails in the preview, **so that** I can diagnose foot sliding
   and desync.

9. **US-15.4.5.1** — **As a** game designer (P-5), **I want** a 1D/2D blend space editor with
   real-time blended output preview, **so that** I can tune locomotion blend parameters.

10. **US-15.4.5.2** — **As a** technical artist (P-13), **I want** drag-based repositioning of
    samples in blend space, **so that** I can adjust speed and direction mapping visually.

11. **US-15.4.6.1** — **As a** game designer (P-5), **I want** a visual state machine editor for
    animation states and transitions with condition expressions, **so that** I can author locomotion
    graphs without code.

12. **US-15.4.6.2** — **As a** technical artist (P-13), **I want** the state machine editor to
    highlight the active state during preview, **so that** I can debug transition issues visually.

13. **US-15.4.7.1** — **As a** technical artist (P-13), **I want** a retargeting editor with
    side-by-side skeleton comparison and bone-pair assignment, **so that** I can share animation
    libraries across different rigs.

14. **US-15.4.7.2** — **As a** game designer (P-5), **I want** real-time retargeted preview,
    **so that** I can verify animations look correct on the target character before committing.
