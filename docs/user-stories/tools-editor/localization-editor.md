# User Stories: Localization Editor

## US-15.13.1 String Table Management

**As a** designer, **I want** to manage all localizable strings in a visual spreadsheet
editor with per-locale columns, translation memory suggestions, and missing-translation
highlighting, **so that** I can track localization coverage without editing raw files.

## US-15.13.2 Locale Preview and Validation

**As a** designer, **I want** to switch the editor to any locale and run a validation pass
that detects text overflow, broken interpolation variables, and RTL layout issues, **so
that** localization defects are caught before they reach players.
