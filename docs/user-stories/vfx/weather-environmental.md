# User Stories — 11.4 Weather & Environmental FX

## US-11.4.1 Experience Dynamic Weather That Transforms the World
**As a** player, **I want** rain with puddles and wet surfaces, snow accumulation with
footprint trails, fog volumes, lightning, wind-driven dust storms, and underwater caustics,
**so that** weather feels alive and changes how the world looks and plays.

| Acceptance criteria | Features | Requirements |
|---|---|---|
| Layered rain with screen droplets and surface ripples | F-11.4.1 | R-11.4.1 |
| Dynamic puddles with material-driven wet response | F-11.4.2 | R-11.4.2 |
| Snow accumulation with deformation trails | F-11.4.3 | R-11.4.3 |
| Localized fog volumes injected into froxel grid | F-11.4.4 | R-11.4.4 |
| Procedural branching lightning with light and audio | F-11.4.5 | R-11.4.5 |
| Wind-coherent particle debris and dust storms | F-11.4.6 | R-11.4.6 |
| Underwater caustics, depth fog, and god rays | F-11.4.7 | R-11.4.7 |
