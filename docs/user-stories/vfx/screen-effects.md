# User Stories — 11.3 Screen Effects

## US-11.3.1 Feel the Impact of Every Explosion and Hit on Screen
**As a** player, **I want** screen shake, motion blur, lens flares, chromatic aberration,
heat haze, and damage overlays that react to gameplay events, **so that** combat feels
visceral and cinematic with immediate visual feedback for every action.

| Acceptance criteria | Features | Requirements |
|---|---|---|
| Perlin-noise screen shake with additive layering | F-11.3.1 | R-11.3.1 |
| Per-object and camera motion blur from velocity buffers | F-11.3.2 | R-11.3.2 |
| Procedural lens flares with temporal smoothing | F-11.3.3 | R-11.3.3 |
| Chromatic aberration, film grain, and vignette | F-11.3.4 | R-11.3.4 |
| Heat haze and refraction from normal maps | F-11.3.5 | R-11.3.5 |
| Damage overlays and directional screen flashes | F-11.3.6 | R-11.3.6 |
| Reduced-motion accessibility option for shake | F-11.3.1 | R-X.9.6 |
