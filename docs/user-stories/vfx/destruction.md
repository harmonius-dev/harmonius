# User Stories — 11.5 Destruction VFX

## US-11.5.1 See Buildings Crumble with Debris, Fire, and Smoke
**As a** player, **I want** debris fragments, dust clouds, sparks, structural cracks, scorch
marks, shockwaves, and spreading fire when structures are destroyed, **so that** siege battles
feel physically believable and every impact has visible consequences.

| Acceptance criteria | Features | Requirements |
|---|---|---|
| Physics-driven debris with material inheritance | F-11.5.1 | R-11.5.1 |
| Material-colored dust clouds and wind-driven smoke | F-11.5.2 | R-11.5.2 |
| Sparks and embers contributing to clustered lighting | F-11.5.3 | R-11.5.3 |
| Progressive crack decals driven by damage accumulation | F-11.5.4 | R-11.5.4 |
| Persistent scorch marks modifying surface properties | F-11.5.5 | R-11.5.5 |
| Expanding shockwave distortion with camera shake | F-11.5.6 | R-11.5.6 |
| Fire propagation driven by wind and material type | F-11.5.7 | R-11.5.7 |
| Global debris budget caps fragments during sieges | F-11.5.1 | R-X.13.1 |
