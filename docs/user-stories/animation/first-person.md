# User Stories — 9.6 First-Person Animation

## US-9.6.1 Feel the Weight of Every Weapon in First Person
**As a** player, **I want** procedural weapon sway, recoil, ADS, and dual wield animations
driven by spring physics with per-weapon parameters, **so that** each weapon feels
physically distinct and responsive without requiring hand-authored sway animations for
every weapon variant.

| Acceptance criteria | Features | Requirements |
|---|---|---|
| Head-bob, landing impact, lean/peek, and tilt | F-9.6.1 | R-9.6.1 |
| Per-weapon spring sway, bob, and sprint tilt | F-9.6.2 | R-9.6.2 |
| Procedural recoil from pattern data with ADS | F-9.6.3 | R-9.6.3 |
| Equip/holster, inspect, and dual wield sequences | F-9.6.4 | R-9.6.4 |
| Separate viewmodel FOV prevents weapon distortion | F-9.6.1 | R-9.6.1 |
| All weapon parameters are data-driven assets | F-9.6.2 | R-X.9.5 |
