# User Stories -- 7.1 Navigation

## US-7.1.1 Configure NavMesh for Different Agent Sizes
**As an** AI designer, **I want** to define NavMesh agent configs (radius, height, step climb,
slope limit) per entity archetype, **so that** humanoids navigate through doorways that
mounts and siege golems cannot fit through.

## US-7.1.2 Navigate Seamlessly Across Streaming Tiles
**As a** player, **I want** NPCs to pathfind across the open world without pausing or rubber-
banding at tile boundaries, **so that** NPC movement feels natural regardless of how far they
travel.

## US-7.1.3 React to Destroyed Buildings
**As an** AI designer, **I want** the NavMesh to rebuild automatically when buildings collapse,
**so that** NPCs discover new shortcuts through rubble and route around debris without manual
navmesh editing.

## US-7.1.4 Navigate Through Player-Built Structures
**As a** player, **I want** NPCs to route around my walls and up my ramps when I build
fortifications, **so that** my base layout has meaningful tactical impact on enemy AI
pathfinding.
