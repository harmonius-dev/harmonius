# User Stories -- 13.12 Character Progression

## Race and Class

### US-13.12.1a Choose a Race for My Character

**As a** player, **I want** to select a race with unique stat modifiers and cosmetic options,
**so that** my character has a distinct identity from creation.

### US-13.12.1b Choose a Class for My Character

**As a** player, **I want** to select a class with a defined ability set and resource type,
**so that** my character has a clear gameplay role from the start.

### US-13.12.1c Switch or Combine Classes

**As a** player, **I want** to switch classes at an NPC or unlock hybrid classes, **so that**
I can adapt my playstyle without restarting.

### US-13.12.1d Prestige at Max Level

**As a** player, **I want** to restart at level 1 with permanent bonuses after reaching max
level, **so that** I have a rewarding end-game loop.

## Talent Trees

### US-13.12.2a Browse the Talent Tree

**As a** player, **I want** to view a visual node-graph talent tree with clear prerequisites
and tier gates, **so that** I can plan my build before spending points.

### US-13.12.2b Allocate and Respec Talent Points

**As a** player, **I want** to spend talent points on unlocked nodes and respec for a
currency cost, **so that** my build choices feel meaningful but not permanent.

### US-13.12.2c Author Talent Trees in the Editor

**As a** designer, **I want** to create and edit talent trees as graph assets in the visual
editor, **so that** I can iterate on talent structures without code changes.

## Professions

### US-13.12.3a Level Up a Profession

**As a** player, **I want** to level a profession independently with recipes unlocking at
skill thresholds, **so that** trade skill progression is its own rewarding path.

### US-13.12.3b Gather Resources with Skill Scaling

**As a** player, **I want** higher gathering skill to yield more and rarer resources, **so
that** investing in a gathering profession feels worthwhile.

### US-13.12.3c Craft Items at a Station

**As a** player, **I want** to craft items at a station using gathered materials and earn
profession XP, **so that** crafting advances my trade skill.

## Achievements

### US-13.12.6a Track Achievement Progress

**As a** player, **I want** to see incremental progress toward achievements as I play, **so
that** I know how close I am to completing each goal.

### US-13.12.6b Receive Achievement Rewards

**As a** player, **I want** to receive items, titles, or currency when I complete an
achievement, **so that** achievements feel rewarding beyond the notification.

### US-13.12.6c See Achievements on My Platform Profile

**As a** player, **I want** engine achievements to sync to Steam, PlayStation, or Xbox,
**so that** my accomplishments appear on my platform profile.

## Item Enhancement

### US-13.12.7 Enchant and Enhance My Equipment

**As a** player, **I want** to enhance my gear through an enchanting system with visible
success rates, material costs, and protection items, **so that** I have a rewarding
long-term goal of improving my equipment beyond its base stats.

## Item Rarity and Affixes

### US-13.12.8a Compare Items by Rarity

**As a** player, **I want** items to have color-coded rarity tiers with stat ranges, **so
that** I can quickly judge item quality at a glance.

### US-13.12.8b Find Items with Random Affixes

**As a** player, **I want** dropped items to have random prefixes and suffixes, **so that**
each drop feels unique and encourages hunting for ideal combinations.

### US-13.12.8c Re-Roll Item Affixes

**As a** player, **I want** to spend currency to re-randomize an item's affixes, **so
that** I have a path to improve a good base item without relying on drops.
