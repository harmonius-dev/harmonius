# User Stories -- Stealth and Cover (13.18)

## US-13.18.1 Stealth Gameplay and Visibility
**As a** player, **I want** my visibility score to depend on lighting, movement, equipment, and
posture so that AI detection range scales accordingly, **so that** I can plan stealth
approaches using environmental awareness and gear choices.

## US-13.18.2 Cover-Based Combat
**As a** player, **I want** to snap to cover, peek to aim, blind-fire, and sprint between cover
points while AI enemies use the same cover system, **so that** firefights involve tactical
positioning rather than standing in the open.

## US-13.18.3 Stealth Takedowns and Distractions
**As a** player, **I want** to perform silent or loud takedowns on unaware enemies and throw
distraction items to lure AI, **so that** I have meaningful stealth tools beyond simply
avoiding detection.
