# User Stories — 13.11 Selection and Picking

## US-13.11.1 Select and Command Units in an RTS

**As a** player, **I want** to box-select groups of units, assign them to control groups, and
issue formation move and attack commands, **so that** I can manage large armies efficiently
in real-time strategy gameplay.

## US-13.11.2 Pick Objects in the 3D World With a Click

**As a** player, **I want** to click on objects in the 3D world and have the correct entity
selected instantly, with interactive objects taking priority over scenery, **so that** I can
interact with the game world responsively and accurately.

## US-13.11.3 Use RTS-Style Selection in a Strategy Game

**As a** player, **I want** box select, control groups, and select-all-of-type to work out of
the box when I choose the RTS preset, **so that** I can manage armies using familiar RTS
conventions without custom setup.

## US-13.11.4 Use Tab-Targeting in an RPG

**As a** player, **I want** to press Tab to cycle through nearby enemies with hostile filtering
when the RPG preset is active, **so that** I can target enemies quickly during combat.

## US-13.11.5 Lock On to Enemies in an Action Game

**As a** player, **I want** auto-targeting and lock-on with camera focus when the action preset
is active, **so that** I can stay focused on my target during fast-paced combat.

## US-13.11.6 Multi-Select With Gizmos in a Builder Game

**As a** player, **I want** to multi-select objects with transform gizmos and group/ungroup
them when the builder preset is active, **so that** I can manipulate complex constructions
efficiently.

## US-13.11.7 Issue Move and Attack Commands to Selected Units

**As a** player, **I want** to issue move, attack, and ability commands to my selected units
through the ability system, **so that** my commands execute consistently regardless of unit
type.

## US-13.11.8 Move Units in Formation

**As a** player, **I want** my selected units to maintain relative positions during group
movement, **so that** my army moves as a cohesive formation rather than a disorganized blob.

## US-13.11.9 Split Selection Into Subgroups

**As a** player, **I want** to split my selection into subgroups and send each to a different
target, **so that** I can execute multi-pronged tactical maneuvers without manually creating
separate control groups.

## US-13.11.10 Undo a Misclicked Command

**As a** player, **I want** to press Ctrl+Z to cancel my last command within a short timeout,
**so that** I can recover from misclicks during fast-paced gameplay.
