# User Stories — 13.1 Gameplay Primitives

## US-13.1.1 Configure Game Mode Rules Without Code

**As a** designer, **I want** to define game mode rules (player limits, scoring, win conditions,
respawn policies) through data assets in the visual editor, **so that** I can create and iterate
on new modes like arenas or dungeons without waiting for a programmer.

## US-13.1.2 Extend the Engine With a Custom Rust Plugin

**As a** game developer, **I want** to build a Rust plugin that registers custom ECS components,
systems, and editor panels, **so that** I can add game-specific functionality without forking
the engine source.
