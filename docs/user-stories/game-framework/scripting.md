# User Stories — 13.4 Gameplay Scripting Integration

## US-13.4.1 Create Gameplay Logic Without Writing Code

**As a** designer, **I want** to author all gameplay logic through visual graphs that access
ECS state, emit events, and trigger abilities, **so that** I can build and iterate on game
mechanics without programming expertise.

## US-13.4.2 Iterate on Gameplay Logic Without Restarting

**As a** designer, **I want** to modify a gameplay logic graph in the editor and see the changes
take effect in the running game within 1 second, **so that** I can rapidly test and tune
mechanics without losing my in-game testing state.
