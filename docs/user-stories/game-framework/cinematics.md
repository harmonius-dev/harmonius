# User Stories — 13.5 Cinematics

## US-13.5.1 Author a Multi-Track Cutscene in the Editor

**As a** designer, **I want** to compose cutscenes using a multi-track timeline with camera,
animation, audio, VFX, and gameplay trigger tracks, **so that** I can create cinematic
sequences visually without writing any code.

## US-13.5.2 Skip a Cutscene Without Missing Rewards

**As a** player, **I want** to skip cutscenes entirely while still receiving all quest rewards
and story progression triggers, **so that** I do not have to rewatch scenes I have already
seen to get my gameplay rewards.

## US-13.5.3 Fast-Forward Through a Cutscene

**As a** player, **I want** to fast-forward cutscenes at 2x or 4x speed while all gameplay
triggers still fire, **so that** I can experience the story quickly without skipping it
entirely.

## US-13.5.4 Pause a Cutscene

**As a** player, **I want** to pause cutscene playback and resume from the exact frame, **so
that** I can attend to real-world interruptions without losing my place in the story.
