# User Stories — 13.9 Inventory System

## US-13.9.1 Organize Items in a Grid Inventory

**As a** player, **I want** to arrange items in a 2D grid inventory with drag-and-drop, auto-sort,
stacking, and splitting, **so that** I can efficiently manage my collected loot and materials.

## US-13.9.2 Trade Items Safely With Other Players

**As a** player, **I want** a mutual-confirmation trade window for player-to-player item and
currency exchange, **so that** I can trade securely without risk of scams or accidental loss.
