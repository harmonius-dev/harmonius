# User Stories -- Weapon Systems (13.16)

## US-13.16.1 Fire Modes and Recoil Mastery
**As a** player, **I want** to toggle between fire modes (semi-auto, burst, full-auto) with
distinct recoil patterns and spread behavior, **so that** I can adapt my weapon handling to
different combat situations and develop skill mastery.

## US-13.16.2 Weapon Customization and Attachments
**As a** player, **I want** to equip attachments (optics, barrels, grips, magazines) to weapon
slots that visually appear on the weapon and modify its stats, **so that** I can personalize my
loadout to match my preferred playstyle.

## US-13.16.3 Ballistic Simulation Configuration
**As a** designer, **I want** to configure per-ammo-type ballistic parameters (gravity drop,
wind deflection, penetration, ricochet) and toggle individual simulation features, **so that**
I can support both realistic tactical shooters and arcade modes from the same system.
