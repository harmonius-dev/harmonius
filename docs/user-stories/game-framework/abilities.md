# User Stories — 13.10 Abilities and Combat

## US-13.10.1 Build Abilities From Reusable Components

**As a** designer, **I want** to compose abilities from modular building blocks (targeting, costs,
effects, VFX, audio) in the visual editor, **so that** I can create hundreds of unique abilities
by remixing shared components without writing any code.

## US-13.10.2 Experience Responsive Melee Combat

**As a** player, **I want** melee attacks to connect exactly when the weapon animation appears to
hit, with per-bone damage regions and satisfying hit-stop feedback, **so that** combat feels
precise and impactful.
