# User Stories -- Turn-Based and Tactical Systems (13.21)

## US-13.21.1 Tactical Grid Combat
**As a** player, **I want** to move units on a tactical grid, spend action points on movement
and abilities in any order, and see reachable cells and attack probabilities before
committing, **so that** I can make informed tactical decisions each turn.

## US-13.21.2 Cover, Flanking, and Overwatch
**As a** player, **I want** grid cover to provide directional defense bonuses that are negated
by flanking, and overwatch to let my units react-fire during the enemy turn, **so that**
positioning and turn-end decisions create meaningful tactical depth.
