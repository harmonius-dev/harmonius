# User Stories -- Game Modes and Systems (13.24)

## US-13.24.1 Wave Defense and Tower Placement
**As a** player, **I want** to place and upgrade towers with configurable targeting modes to
defend against data-driven enemy waves with escalating difficulty, **so that** I can engage in
strategic tower defense gameplay with meaningful upgrade paths.

## US-13.24.2 Game Feel and Feedback Polish
**As a** designer, **I want** to author composable feedback stacks (camera shake, freeze frame,
haptics, particle bursts, time scale ramps) triggered by gameplay events without code, **so
that** I can polish game feel and impact through visual authoring alone.

## US-13.24.3 Fast Travel and Respawn
**As a** player, **I want** to discover named waypoints through exploration and fast travel
between them, and respawn at the nearest graveyard on death with optional corpse-run
mechanics, **so that** world navigation and death penalties feel fair and structured.
