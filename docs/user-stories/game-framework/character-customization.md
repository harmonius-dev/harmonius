# User Stories — 13.8 Character Customization

## US-13.8.1 Create a Unique-Looking Character

**As a** player, **I want** to customize my character's face, body, skin, hair, and eyes with
fine-grained sliders and preset blending, **so that** my character looks distinct from every
other player's character.

## US-13.8.2 See Equipment Fit My Character's Body Shape

**As a** player, **I want** equipped armor and clothing to automatically conform to my
character's unique body proportions without clipping, **so that** my visual customization
choices are preserved regardless of what gear I wear.
