# User Stories -- Fog of War (13.20)

## US-13.20.1 Strategic Vision and Map Reveal
**As a** player, **I want** unexplored areas hidden by fog, previously seen areas showing
last-known state as ghosts, and currently visible areas rendered in real time, **so that**
scouting and map control are meaningful strategic decisions.

## US-13.20.2 Vision Modifiers and Tactical Terrain
**As a** designer, **I want** to place vision modifier volumes (stealth bushes, smoke, high
ground, darkness zones) that alter fog-of-war behavior dynamically, **so that** maps have
tactically interesting terrain features that reward positioning.
