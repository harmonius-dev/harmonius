# User Stories -- 3D Camera System (13.25)

## US-13.25.1 Third-Person Camera with Collision Avoidance
**As a** game developer, **I want** to configure third-person cameras with orbital follow,
shoulder offset, spring arm collision, and deoccluder/decollider extensions, **so that**
the camera never clips through geometry or loses sight of the player character.

## US-13.25.2 Data-Driven Camera Switching and Blending
**As a** designer, **I want** priority-based virtual cameras that blend automatically on
gameplay state changes (combat, climbing, mounting, cutscene) using configurable curve types
and per-camera-pair rules, **so that** camera transitions feel cinematic without manual
scripting per scenario.
