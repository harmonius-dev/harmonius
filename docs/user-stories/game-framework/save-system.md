# User Stories — 13.3 Save System

## US-13.3.1 Resume Progress After a Crash

**As a** player, **I want** my progress to be automatically saved at checkpoints and time
intervals, **so that** I lose minimal progress if the game crashes or I need to quit
unexpectedly.

## US-13.3.2 Continue Playing on a Different Device

**As a** player, **I want** my save data to sync to the cloud automatically, **so that** I can
pick up where I left off on a different device without manually transferring files.
