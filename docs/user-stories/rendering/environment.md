# Environment and Atmosphere User Stories

## US-2.7.1 Dynamic Weather and Time of Day

**As a** designer, **I want** a weather state machine that drives clouds, fog, precipitation,
lighting, and material wetness with smooth transitions between states, **so that** I can
create immersive outdoor environments with dynamic weather cycles.

## US-2.7.2 Ocean and Water Authoring

**As a** artist, **I want** an FFT-based ocean with art-directable breaking waves controlled
by guide curves, **so that** I can create realistic shoreline environments with full artistic
control over wave shape, timing, and animation.
