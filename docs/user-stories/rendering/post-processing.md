# Post-Processing User Stories

## US-2.9.1 Cinematic Camera Effects

**As a** artist, **I want** depth of field driven by real camera parameters (aperture, focal
length, focus distance) with circular bokeh, motion blur, and bloom, **so that** cutscenes
and gameplay cameras achieve a cinematic film look.

## US-2.9.2 Custom Visual Effects

**As a** technical artist, **I want** to create custom fullscreen post-process materials that
read scene textures (depth, normals, velocity) and output arbitrary color transformations,
**so that** I can implement game-specific visual effects without modifying the engine.
