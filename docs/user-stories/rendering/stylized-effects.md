# Stylized and Gameplay Effects User Stories

## US-2.11.1 Gameplay Object Highlighting

**As a** designer, **I want** per-entity outlines, glow highlights, and x-ray silhouettes
driven by ECS components, **so that** gameplay systems can control visual feedback (selection,
hover, loot rarity, enemy tracking) without custom rendering code.

## US-2.11.2 Isometric Interior Visibility

**As a** designer, **I want** automatic roof fading and cut-through visibility with
configurable fade modes (dithered, dissolve, alpha) when the camera or player enters a
structure, **so that** isometric and top-down games show interior spaces clearly.
